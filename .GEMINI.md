# CLAUDE.md — DCLG Toy Experiment Implementation Guide (Updated)

## 1. 프로젝트 목표 및 현황

IP-Adapter 또는 텍스트 기반 다중 객체 생성에서, 각 객체의 Cross-Attention 맵을 공간적으로 격리하여 **키메라(Chimera) 현상을 억제**하는 **DCLG (Decoupled Cross-attention Latent Guidance)** 기법을 구현하고 검증한다.

- **현재 상태:** 텍스트 기반(Text-only) 가이던스 코어 로직 완벽 검증 완료.
- **주요 성과:** 텍스트 토큰 간의 공간적 배타성(Exclusivity)을 유도하는 수학적 프레임워크 정립.

## 2. 핵심 업데이트 사항 (Lessons Learned)

### 2-1. 수학적 안정성 (Stability)
- **Precision:** `float16`에서의 Gradient Underflow 문제를 해결하기 위해 전 과정을 **`float32`**로 수행.
- **Guidance Target:** Latent($x_t$)를 직접 수정하던 방식에서 **Noise Prediction($\epsilon_\theta$) 수정 방식**으로 변경.
  - 수식: $\epsilon_{updated} = \epsilon_{out} + \lambda \cdot \nabla_{x_t} Loss$ (디노이징 방향을 보존하며 가이던스 주입)
- **Normalization:** 레이어 해상도와 관계없이 일정한 그래디언트 세기를 유지하기 위해 모든 Loss 연산에 **`mean()`** 정규화 적용.

### 2-2. 배타성 강화 로직 (Exclusivity)
- **Cosine Similarity:** 단순히 맵을 곱하는 대신, 두 어텐션 분포의 **코사인 유사도를 최소화**하여 서로 직교(Orthogonal)하도록 강제.
- **Attention Sharpening:** 어텐션 맵에 **제곱($map^2$)**을 취해 낮은 활성화 영역은 죽이고 핵심 영역의 경계를 선명하게 만듦.
- **Erasure Penalty:** 특정 객체가 화면에서 지워지는 것을 방지하기 위해 각 맵의 **Max 활성도가 일정 수준(0.5) 이상 유지**되도록 페널티 부여.

### 2-3. 시각화 및 분석 (Grounding Analysis)
- **Overlay Visualization:** 원본 이미지 위에 기사(Red)와 오크(Blue)의 어텐션 영역을 겹쳐서 표시하여 **텍스트-비주얼 연결성(Grounding)**을 정밀하게 검증.
- **Multi-step Debugging:** Step 25, 15, 5 등 디노이징 초기/중기 과정의 맵을 보존하여 구도가 잡히는 과정을 추적.

## 3. 최신 구현 명세

### 3-1. hook_manager.py (Text Attention Hook)
- `up_blocks` 내의 `attn2` 레이어에 `SaveAttnProcessor`를 주입하여 77개 텍스트 토큰의 어텐션 맵을 실시간 캡처.

### 3-2. dclg_pipeline.py (Core Logic)
```python
def compute_chimera_loss(map_A, map_B):
    # 1. Sharpening
    map_A, map_B = map_A**2, map_B**2
    # 2. Normalization
    map_A = map_A / map_A.max(); map_B = map_B / map_B.max()
    # 3. Exclusivity (Cosine Similarity)
    cos_sim = F.cosine_similarity(map_A.view(-1), map_B.view(-1), dim=0)
    # 4. Erasure Penalty
    penalty = relu(0.5 - map_A.max()) + relu(0.5 - map_B.max())
    return cos_sim + penalty
```

### 3-3. run_dclg.py (Experiment Entry)
- 프롬프트에서 타겟 단어(`knight`, `orc`)의 인덱스를 자동 추출하여 가이던스 대상으로 설정.
- $\lambda$ 스윕 범위: `[0.0, 10.0, 50.0, 100.0, 200.0, 300.0]` (Noise Pred 수정 방식에 최적화된 스케일).

## 4. 향후 계획 (Next Step: IP-Adapter)

- **Motivation:** 텍스트 어텐션은 전역적 정보가 많아 공간 분리 신호가 희소함.
- **Action:** IP-Adapter의 **Image Cross-Attention**을 활용하여 더 강력하고 구체적인 공간 가이던스 신호원을 확보.
- **Architecture:** 텍스트와 이미지 임베딩이 분리된 **Decoupled Attention** 구조를 엄격히 준수하며 가이던스 적용.
